# Todo List App

A simple WPF application for managing a todo list with local storage.

## Features

- Add new todos with title and description
- Update existing todos
- Remove todos
- Mark todos as completed
- Data is stored locally in a JSON file

## How to Run

1. Ensure you have .NET 10 installed.
2. Navigate to the TodoApp directory.
3. Run `dotnet run`

## UI

The app has a clean, modern UI with:
- Input fields for title and description
- Buttons for add, update, and remove operations
- A list view displaying all todos with completion status and creation date
